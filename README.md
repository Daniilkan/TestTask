# TestTask

**TestTask** — проект для парсинга товаров с маркетплейса, их классификации и аналитики. В проекте используется Selenium для парсинга, Pydantic для валидации данных, а также реализована интеграция с CRM-системой.

## Структура проекта

- **app/** — основная директория приложения:
  - `services/` — сервисы для парсинга, классификации, аналитики и интеграции с CRM.
  - `models/` — модели данных (например, `Product`).
  - `clients/` — клиенты для работы с внешними API (например, CRM и LLM).
  - `config.py` — файл конфигурации проекта.
- **tests/** — директория с тестами.
- **.env** — файл с переменными окружения.

## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone https://github.com/your-repository/TestTask.git
cd TestTask
```

### 2. Установка зависимостей

Убедитесь, что установлен Python версии 3.9 или выше и создано виртуальное окружение:

```bash
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
venv\Scripts\activate     # Для Windows
pip install -r requirements.txt
```

### 3. Настройка переменных окружения

Скопируйте файл `.env.example` в `.env` и заполните необходимые переменные:

```bash
cp .env.example .env
```

### 4. Запуск приложения

```bash
python app/main.py
```

### 5. Запуск тестов

```bash
pytest
```

## Возможности проекта

- **Парсинг** — сбор информации о товарах с маркетплейса Ozon с использованием Selenium.
- **Классификация** — определение ценового сегмента товаров (Эконом, Стандарт, Премиум) с помощью LLM.
- **Аналитика** — поиск топ-5 самых дорогих товаров сегмента «Премиум» и топ-5 самых дешёвых товаров сегмента «Эконом».
- **Интеграция с CRM** — автоматическое создание задач в CRM с результатами аналитики.
